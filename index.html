Me

I got into computers when I was in 5th grade. We had computer class that was a different form of learning. You could do lesson in math reading and other subjects interactly with a program. This really intrigued me so instead of going to lunch I would grab some food eat quickly and then head to the computer room to help Mr. Dudka load up the lessons. At that day and age the data was on 2MB 2.5" "floppy" disks. You had to load 8-10 of them for one program to run. From there I taught myself how to code on TI80 calculators. Being the nerdy guy I was I create apps that would compute math equations with inputs. When I reached highschool I was able to attend out computer scient class where I learned some BASIC, Pascal, C++. Life happened and I picked up with some self learning and started with a bit of Java and some JavaScript. It became apparent that I needed to learn more and learn it faster so I enrolled in Lambda School's Part-Time Full-Stack Web Development & Computer Science program. Every single day I learn more and more and hope to put out some great code. I am looking forward to doing what I love for work. Be gentle as this is a work in progress.
